Análise de dados: Orçamento de redes sociais x vendas¶

Este um projeto que envolve machine learning para anlise de dados de marketing, oramentos(budget) de redes sociais e a relao com as vendas geradas, os dados foram disponibilizados na plataforma Kaggle.

Sobre o conjunto de dados ser adota a seguinte interpretao, a cada rede/mdia social das colunas 'youtube', 'facebook' e 'newspaper' destinado um oramento(budget) de marketing, sendo a que coluna 'sales' representa as vendas, o retorno do investimento em marketing.

📃 Demanda da análise¶

  • Analisar a correlao dos oramentos(budgets) com as vendas(sales)
  • Gerar um modelo preditivo para relao de oramento e vendas

📋 Tópicos da análise¶

  1. Anlise das medidas estatsticas dos dados
  2. Distribuio do budget entre as mdias e os valores de vendas
  3. Correlao entre as variveis
  4. Criao do modelo de regresso linear mltipla 
     1. Preparao dos dados de treino e teste 
 2. Treinamento do modelo
 3. Predies do modelo com dados de treino e teste
 4. Mtricas de performance
 5. Previses com novos dados

def graf_histplot(param1, param2, param3, param4='royalblue'):
    sns.histplot(data=param1, ax=param3, color=param4, kde=True)
    param3.set_ylabel('')
    param3.set_xlabel('')
    param3.tick_params(labelsize=15)
    param3.set_title(param2, loc='right', fontsize=15)

# features, variveis independentes
X = df.drop(['sales'], axis=1)

# varivel resposta, varivel dependente

seed = 10

# separao dos dados de treino e teste
x_train, x_test, y_train, y_test = train_test_split(
    X, y, test_size=perc, random_state=seed)

4. Métricas de performance¶

# treino - r2, rmse, mae e mape
r2_train = r2_score(y_train, pred_train) * 100
rmse_train = np.sqrt(mean_squared_error(y_train, pred_train))
mae_train = mean_absolute_error(y_train, pred_train)
mape_train = mean_absolute_percentage_error(y_train, pred_train)

# teste - r2, rmse, mae e mape
r2_test = r2_score(y_test, pred_test) * 100
rmse_test = np.sqrt(mean_squared_error(y_test, pred_test))
mae_test = mean_absolute_error(y_test, pred_test)
mape_test = mean_absolute_percentage_error(y_test, pred_test)

data = {
    '': ['Treino', 'Teste'],
    'R^2': [r2_train, r2_test],
    'RMSE': [rmse_train, rmse_test],
    'MAE': [mae_train, mae_test],
    'MAPE': [mape_train, mape_test]
}

print(tabulate(data, headers='keys', tablefmt='fancy_outline',
